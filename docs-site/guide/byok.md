# Bring Your Own Key (BYOK)

BYOK lets your organization use **your own LLM provider key** (OpenAI, Azure OpenAI, Anthropic,
Gemini). Billing, the data-processing agreement, and model access stay in **your** account.

Configure it under **Admin → LLM Config** (write-only key, encrypted at rest). If no key is set,
the platform default is used.
