# Team Memory (Mem0)

ARIA remembers context at two levels — **per user** and **per team** — using Mem0 (vectors in Qdrant).

- **User memory:** your preferences and recent context.
- **Team memory:** shared conventions and business rules that apply to everyone on the team
  (human-approved by a team lead).

Admins can view, edit and delete entries under **Admin → Memory**.

See the screen guide: [Admin · Memory](./admin-memory.md).
