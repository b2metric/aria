### 2. Embedding Kolon Tipleri — KALDIRILDI

**Karar:** pgvector KULLANILMAZ. Embedding'ler Qdrant'ta (Mem0 vector backend). PostgreSQL sadece structured metadata (memory_entries.content JSONB, vault_knowhow.description vs). `memory_entries.embedding` ve `vault_knowhow.embedding` kolonlari KALDIRILDI.

- Qdrant: vector storage + semantic search (1536-d openai, 768-d ollama)
- PostgreSQL: structured metadata, CRUD, approval workflow, audit log

### 3. Enum Isimlendirme

- `job_status` eksik durum: 'cancelled'?
