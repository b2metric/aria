Vault Auto-Sync

Keep your Markdown schemas synchronized with live databases automatically.

ARIA reads schemas from local Markdown files stored in docs/vaults/. This gives you version control over descriptions, keywords, and manual relationships.

How It Works

The Auto-Sync engine connects to your live customer database (Oracle, PostgreSQL, MySQL) using credentials managed via the Tenant Config.

It reads information_schema and performs a smart diff against the local Markdown files. It then carefully merges new columns without overwriting your human-edited tags and descriptions.

⚠️ Decrypted Safely

Database passwords are saved using AES-256 (Fernet) encryption. During an auto-sync, ARIA decrypts the password securely in-memory to establish a temporary connection.

Triggering a Sync

  • Manually: Navigate to the Data Dictionary under the Admin panel and click the Re-Sync button.
  • Cron / Background Task: (Coming Soon) ARIA can be scheduled to run the sync every night via Prefect.
